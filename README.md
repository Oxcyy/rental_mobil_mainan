## рҹ“– Tentang Aplikasi

### вҡҷпёҸ State Management вҖ” Provider

Aplikasi menggunakan **Provider** dengan dua `ChangeNotifier` utama:

```dart
MultiProvider(
  providers: [
    ChangeNotifierProvider(create: (_) => AppStore()),      // data mobil & rental
    ChangeNotifierProvider(create: (_) => ThemeProvider()), // dark/light mode
  ],
  child: const MyApp(),
)
```

| Method | Kegunaan |
|--------|----------|
| `context.watch<T>()` | Rebuild otomatis saat state berubah |
| `context.read<T>()` | Akses state tanpa rebuild |
| `notifyListeners()` | Trigger update ke semua listener |

### рҹ—„пёҸ Supabase Integration

<div align="center">

| Layanan | Fungsi |
|:-------:|--------|
| рҹ”‘ **Auth** | Login email & password, pantau session via stream |
| рҹ—ғпёҸ **Database** | Tabel `cars` dan `rentals` di PostgreSQL |
| рҹ“Ұ **Storage** | Upload & akses foto mobil di bucket `car_images` |

</div>

```dart
// Semua operasi dipusatkan di SupabaseService
static Future<String> uploadCarImage(File imageFile) async {
  final fileName = 'car_${DateTime.now().millisecondsSinceEpoch}.jpg';
  await _client.storage.from('car_images').upload(fileName, imageFile);
  return _client.storage.from('car_images').getPublicUrl(fileName);
}
```

### рҹ”’ Konfigurasi `.env`

Kredensial API disimpan aman menggunakan **`flutter_dotenv`** вҖ” **tidak di-push ke GitHub**.

```env
# .env  (jangan di-commit!)

SUPABASE_ANON_KEY=your-anon-key-here
```

```dart
// main.dart
await dotenv.load(fileName: '.env');
await Supabase.initialize(
  url: dotenv.env['SUPABASE_URL']!,
  anonKey: dotenv.env['SUPABASE_ANON_KEY']!,
```

## рҹҡҖ Cara Instalasi

### Prasyarat

- Flutter SDK `>=3.0.0 <4.0.0`
- Android Studio / VS Code
- Akun Supabase aktif

### Langkah-langkah

```bash
# 1. Clone repositori
git clone https://github.com/Oxcyy/rental_mobil_mainan.git
cd rental_mobil_mainan

# 2. Buat file .env
cp .env.example .env
# вҶ’ Edit .env dan isi SUPABASE_URL & SUPABASE_ANON_KEY

# 3. Install dependencies
flutter pub get

# 4. Jalankan aplikasi
flutter run
```

> рҹ’Ў Dapatkan `SUPABASE_URL` dan `SUPABASE_ANON_KEY` di **Supabase Dashboard вҶ’ Project Settings вҶ’ API**
